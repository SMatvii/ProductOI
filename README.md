# ProductOI - Сервіс доставки їжі

[English version](README_en.md)

Сучасна веб-платформа для замовлення доставки їжі 24/7 з чистим дизайном у білому та фіолетовому кольорах.

## Особливості

- ✅ **Каталог товарів** - широкий вибір страв розділених по категоріям
- ✅ **Кошик** - легко додавайте та видаляйте товари
- ✅ **Доставка** - експрес, стандартна доставка або самовивіз
- ✅ **Оплата** - карта, готівка, Apple Pay, Google Pay
- ✅ **Адаптивний дизайн** - красиво виглядає на всіх пристроях
- ✅ **Інтерактивний** - живий кошик, фільтрація по категоріям

## Технологічний стек

- **Backend**: Python Flask
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Design**: Білий та фіолетовий колір, чистий UI/UX

## Структура проекту

# ProductOI — доставка їжі

Короткий демонстраційний магазин для замовлення їжі: каталог, кошик і оформлення замовлення.

![Проект](app/static/images/product.png)

Ключові бібліотеки
- Python: `Flask==2.3.2`, `Werkzeug==2.3.6`, `Jinja2==3.1.2`
- Frontend: HTML5, CSS3, ванільний JavaScript

Швидкий старт
1. Встановити залежності:
```bash
pip install -r requirements.txt
```
2. Запустити сервер:
```bash
python run.py
```
3. Відкрити: `http://localhost:5000`

Коротко про функції
- Каталог товарів з категоріями
- Кошик з редагуванням кількості та видаленням
- Оформлення з вибором доставки та оплати

Структура (коротко)
```
app/
  templates/   # HTML-шаблони
  static/      # css, js, images
run.py
requirements.txt
```
